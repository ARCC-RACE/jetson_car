Ausführliche Beschreibung

Klasse für den Zugriff auf das GNUBLIN MODULE-DISPLAY 2x16.

Diese Klasse ermöglicht das Beschrieben des DOGM-Displays.

Definiert in Zeile 20 der Datei module_dogm.h.

Setzt die standard RS-Pins.

Standard RS-Pin: GNUBLIN: GPIO14 RASPBERRY PI: GPIO4 BEAGLEBONE_BLACK: GPIO60
