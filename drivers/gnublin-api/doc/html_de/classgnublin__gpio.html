Ausführliche Beschreibung

Klasse für den zugriff auf die GPIO Pins.

Mit der gnublin_gpio API lassen sich die GPIO-Ports auf dem GNUBLIN einfach aus dem eigenem Programm heraus ansteuern.

int gnublin_gpio::pinMode ( int  pin,
std::string  direction 
)

Pin Modi festlegen.

Mit dieser Funktion kann der Modi des Pins festgelegt werden, also ob er als Eingang oder Ausgang genutzt werden soll.
Vor allen anderen Zugriffen muss diese Funktion ausgeführt werden.

Hinweis: GPIO 4 kann auf dem GNUBLIN board nur als Ausgang genutzt werden.

pinNummer des GPIO-Pins
