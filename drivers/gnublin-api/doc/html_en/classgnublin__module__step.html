Detailed Description

Class for accessing GNUBLIN module-step.

The GNUBLIN Module-step can be easily controlled with the gnublin_step API. The Module uses the I2C-Bus.

int gnublin_module_step::drive ( int  steps)

Drive.

This Funktion reads the actual position from the motor and adds the amount of given steps to drive. So you can let the motor drive an amount of steps, without heaving trouble with the absolute positions. If the amount of given steps is higher as 32760 the movement has to be seperated in two or more single movements to avoid an overflow of the register on the TMC222. The process will be blocked until the Limit has been undershot.

Returns
success: 1, failure: -1

int gnublin_module_step::getMotionStatus ( )

Get motion status.

This funktion Indicates the actual behavior of the position controller.
0: Actual Position = Target Position; Velocity= 0
1: Positive Acceleration; Velocity > 0
2: Negative Acceleration; Velocity > 0
3: Acceleration = 0 Velocity = maximum pos Velocity
4: Actual Position /= Target Position; Velocity= 0
5: Positive Acceleration; Velocity < 0
6: Positive Acceleration; Velocity < 0
7: Acceleration = 0 Velocity = maximum neg Velocity
-1: failure

Returns
motionStatus
